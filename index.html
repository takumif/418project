Summary

We will write an efficient, parallel shakashaka solver using JavaScript Web Workers.

Background

(Shakashaka)[https://en.wikipedia.org/wiki/Shakashaka] is a logic puzzle, in which you fill a grid of squares with triangles to form rectangles. It is similar to nonograms in concept, and is known to be NP-complete.
